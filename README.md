# Alibaba-website-training-project
پیاده‌سازی دقیق، تمیز و کاملاً ریسپانسیو صفحه Seller Help Center علی‌بابا با استفاده از HTML و CSS مدرن، با تمرکز بر ساختار اصولی و UI حرفه‌ای.
⚠️ This project focuses purely on UI implementation.
JavaScript logic and backend interactions are intentionally excluded to showcase HTML & CSS mastery.

«می‌دونم JS چیه، ولی اینجا هدفم چیز دیگه‌ست.»
